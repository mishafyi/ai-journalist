# Security Policy

## Supported versions

This project is pre-1.0. Security fixes are made against the latest `0.x`
release on the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| 0.x     | :white_check_mark: |

## Reporting a vulnerability

Please report security issues **privately** — do not open a public issue for a
vulnerability.

Use GitHub Security Advisories: go to the repository's
[**Security** tab](https://github.com/mishafyi/ai-journalist/security) and click
**"Report a vulnerability"**. This opens a private advisory visible only to the
maintainers, where we can discuss and coordinate a fix.

We'll acknowledge your report and keep you updated on remediation and disclosure
timing. Thank you for helping keep AI Journalist and its users safe.
